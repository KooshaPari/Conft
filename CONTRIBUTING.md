# Contributing to Conft

Thanks for your interest in contributing.

## Workflow

1. Fork or branch from `main`.
2. Use feature branches: `feat/<topic>`, `fix/<topic>`, `chore/<topic>`.
3. Run local quality checks before opening a PR.
4. Open a PR against `main`; CI must pass.

## Commit Messages

Use Conventional Commits: `feat:`, `fix:`, `chore:`, `docs:`, `refactor:`, `test:`.

## Code of Conduct

Be respectful. Harassment or abuse will not be tolerated.
